Projects

Recipe-Ratings-Prediction-Model

Web Application
Skills Used: Pandas, Python, ML(Decision tree, Random Forest classifier)

The project focused on building a Recipe Ratings Prediction Model. The goal was to predict the ratings of recipes by extracting features from review data. The evaluation metric used was RMSE (Root Mean Square Error), which measures the difference between predicted and actual ratings. The project started with a baseline model using nutrition-related features and then evolved to include additional quantitative features like minutes, n_steps, calories, n_ingredients, reviews, and description. Hyperparameters were optimized using techniques like Polynomial Features and Grid Search CV. Different models were evaluated, including linear regression, decision trees, and random forest classifiers. The final model selected was the Random Forest Classifier due to its high R^2 and precision scores.

Skills Used

  • Data Analysis: Conducted exploratory data analysis to understand the dataset and identify relevant features for prediction.
  • Feature Engineering: Extracted quantitative features from the review data, such as minutes, n_steps, calories, n_ingredients, reviews, and description.
  • Machine Learning: Built and evaluated regression models, including linear regression, decision trees, and random forest classifiers.
  • Hyperparameter Tuning: Utilized techniques like Polynomial Features and Grid Search CV to optimize hyperparameters for the models.
  • Statistical Analysis: Performed a fairness analysis using a permutation test to compare the precision of the model for recipes with different cooking times.
  • Model Evaluation: Used RMSE, R^2, and precision scores to assess and compare the performance of different models.
  • Conclusion Drawing: Analyzed the results of the model evaluation and made conclusions based on observed differences and statistical tests.
